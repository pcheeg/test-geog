# GeoRank Countries Dataset v0.2

Playable countries: 191

This version adds:
- coastlineKm
- highestPointM
- highestPointName

Rules:
- UN member states only
- Territories excluded
- South Africa and Bolivia excluded due to multiple-capital ambiguity
- Dataset is frozen for game consistency

Source notes:
- Coastline: CIA World Factbook coastline_wf via LSYS/country-coastline-distance.
- Highest point: Flagpedia highest-point list and Wikipedia elevation-extremes table, frozen as game values.

Caution:
Coastline figures differ by source. Do not mix sources inside gameplay.
